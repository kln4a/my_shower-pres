Flexbox

  1. 2008 - обсуждение “Flexible Box Model” (XUL & XAML)
  2. 2009 - черновик “Flexible Box Layout Module” (flexbox 2009)
  3. 2011 - Tab Atkins публикует 2 черновика (flexbox 2011)
  4. 2012 - Candidate Recommendation (flexbox 2012)
  5. 2014 - поддержка всеми новыми браузерами (IE11 тоже)
